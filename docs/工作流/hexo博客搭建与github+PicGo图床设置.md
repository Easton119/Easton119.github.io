简介：使用的Hexo搭建博客，Typora本地笔记，Typora设置PicGo插件进行上传，然后在PicGo里面选择github来作为免费图床。

注意，要使用`npm install hexo-asset-img --save` 下载img插件到文件夹中，不然就算上传到图床成功也在博客网页中显示不出来。

>hexo使用指令：
	`hexo clean` 清除原有public
	`hexo g` 生成
	`hexo d` 发布
